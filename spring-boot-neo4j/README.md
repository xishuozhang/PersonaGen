## SpringBoot 整合 Neo4j

> 首次发布 2022年9月
>

简单整合 Neo4j 相关操作
```
├
└─src
   ├─main
   │  ├─java
   │  │  └─com
   │  │      └─liuyuncen
   │  │          ├─config               // 配置文件
   │  │          ├─controller           // 控制层
   │  │          ├─entity               // 实体层
   │  │          └─repository           // dao 层
   │  └─resources
   └─test
       └─java
           └─com
               └─liuyuncen
```
